genlevel
About

A self-contained generative synthesizer written in Rust, compiled to WebAssembly. Four layers of evolving sound — slow bass drones, filtered mid-range oscillators, reverbed highs, and stochastic plucked strings — create a unique ambient soundscape on each play.

Every run is seeded randomly. No two listens are the same.
